【方案 1】构建 Linux 版本（最简单）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开终端，进入此目录：
   cd /path/to/utils

2. 运行构建脚本：
   ./构建Linux应用.sh

3. 等待 2-5 分钟

4. 完成！得到：dist/期刊文本提取工具

5. 使用：
   ./dist/期刊文本提取工具
   或双击运行


【方案 2】构建 Windows .exe（用 Docker）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

前提：已安装 Docker

1. 安装 Docker（如未安装）：
   sudo apt update
   sudo apt install docker.io
   sudo systemctl start docker
   sudo usermod -aG docker $USER

   然后重新登录或运行：newgrp docker

2. 运行构建脚本：
   ./构建Windows应用_Docker.sh

3. 首次运行会下载 Docker 镜像（约 500MB）

4. 等待 5-10 分钟

5. 完成！得到：dist/期刊文本提取工具.exe

6. 将 .exe 文件复制到 Windows 电脑使用

【完整流程 - 构建 Windows .exe】

# 1. 安装 Docker
sudo apt update
sudo apt install docker.io
sudo systemctl enable docker
sudo systemctl start docker
sudo usermod -aG docker $USER

# 2. 重新登录（重要！）
logout
# 或
newgrp docker

# 3. 进入目录
cd /Users/alpha/Downloads/assistant/utils

# 4. 运行构建脚本
./构建Windows应用_Docker.sh

# 5. 等待完成

# 6. 检查输出
ls -lh dist/期刊文本提取工具.exe

# 7. 复制到 U 盘或上传到网盘
cp dist/期刊文本提取工具.exe /path/to/usb/


【完整流程 - 构建 Linux 版本】

# 1. 安装依赖
sudo apt update
sudo apt install python3 python3-pip python3-tk

# 2. 进入目录
cd /Users/alpha/Downloads/assistant/utils

# 3. 运行构建脚本
./构建Linux应用.sh

# 4. 等待完成

# 5. 测试运行
./dist/期刊文本提取工具

═══════════════════════════════════════════════════════════════════
  故障排除
═══════════════════════════════════════════════════════════════════

❌ "./构建xxx.sh: Permission denied"
   → 运行：chmod +x 构建*.sh

❌ "docker: command not found"
   → 安装 Docker：sudo apt install docker.io

❌ "permission denied while trying to connect to Docker"
   → 运行：sudo usermod -aG docker $USER
   → 然后重新登录

❌ "Cannot connect to the Docker daemon"
   → 启动 Docker：sudo systemctl start docker

❌ 构建失败
   → 检查网络连接（需要下载依赖）
   → 确保有足够磁盘空间（至少 2GB）
